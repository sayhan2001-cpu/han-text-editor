# han-text-editor
Trying to create a simple text editor in C as part of my project
to improve my expertise in C language (after learning it for a month).
I am experimenting with different data structures to store text,
such as arrays (worst), gap buffers, ropes and piece table(best).

I hope this project would come into fruitation and I can improve upon it.

So the labelling of my text editor version is like this:
The numbering means the type of data structures I am using. The
letter besides the number means the version updates I will make.

Example:
1 means array
2 means gap buffer
3 means piece table
4 means rope

Initially, I would just save the file as 1 or 2 when I am first implementing a data structure. Then later if I make small changes using that data structure, I would make the file 1a. followed by 1b, 1c and so on (you get the gist).
